# powerbi-sales-dashboard
This Power BI project analyzes sales data to identify revenue trends, top-performing products, and regional performance. The dataset was cleaned using Power Query, and key KPIs like Total Sales, Total Orders, and Average Order Value were created using DAX. Interactive slicers enable dynamic analysis by region and time.

# Power BI Sales Dashboard

## Overview
This project analyzes sales data using Power BI to understand revenue trends, top-selling products, and regional performance.

## Dataset
- Source: Public sales dataset
- Records: 100 sales orders
- Key fields: Order Date, Region, Item Type, Total Revenue, Total Profit

## Key Insights
- Total Sales and Total Orders KPIs
- Monthly sales trend
- Top 5 item types by revenue
- Interactive slicers for region/year analysis

## Tools Used
- Power BI Desktop
- Power Query (data cleaning)
- DAX (KPIs: Total Orders, Average Order Value)

## How to Use
Download the `.pbix` file and open it in Power BI Desktop to explore the dashboard.
